# BinarySearchTree
C# Minimal AVL Tree Implementation

This is just a basic implementation of an AVL Binary Search Tree. 
NOTE: This is not production code, just an excercise in implementing an algorithm.

# Right Right Case #

     3                  5
      \                / \
       5    -->       3   7
        \              
         7              

#Right Left Case #

     3                  3                  4
      \                  \                / \
       5    -->           4    -->       3   5            
      /                    \
     4                      5
